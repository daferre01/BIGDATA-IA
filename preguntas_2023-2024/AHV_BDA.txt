BDA

1. ¿Cuál es el tipo de base de datos que representa MongoDB?
   - A) Relacional
   - B) No relacional
   - C) SQL

   Respuesta Correcta: B) No relacional

2. ¿Qué lenguaje se utiliza principalmente para escribir consultas en Hive?
   - A) SQL
   - B) Java
   - C) Python

   Respuesta Correcta: A) SQL

3.  ¿Cuál es el propósito principal de Apache Flume?
   - A) Procesamiento de datos en tiempo real
   - B) Recopilación y agregación de datos de registro
   - C) Almacenamiento distribuido de datos

   Respuesta Correcta: B) Recopilación y agregación de datos de registro

4. ¿Qué rol cumple Kafka en la arquitectura de procesamiento de datos?
   - A) Base de datos relacional
   - B) Sistema de mensajería distribuida
   - C) Herramienta de procesamiento de consultas

   Respuesta Correcta: B) Sistema de mensajería distribuida

5. ¿Qué significa HDFS en el contexto de Hadoop?
   - A) Hadoop Distributed File System
   - B) High Data File Storage
   - C) Hadoop Data File Server

   Respuesta Correcta: A) Hadoop Distributed File System

6.  ¿Cómo se representan los datos en MongoDB?
   - A) En tablas
   - B) En colecciones y documentos
   - C) En filas y columnas

   Respuesta Correcta: B) En colecciones y documentos

7. ¿Cuál es el componente principal de Hive que realiza la traducción de consultas SQL a tareas MapReduce?
   - A) Metastore
   - B) Hive CLI
   - C) Hive Query Compiler

   Respuesta Correcta: C) Hive Query Compiler

8.  ¿Cuál es el agente principal en Apache Flume?
   - A) Collector
   - B) Source
   - C) Channel

   Respuesta Correcta: B) Source

9.  ¿Cuál es el término utilizado para describir un grupo de servidores Kafka que trabajan juntos?
   - A) Grupo de consumidores
   - B) Clúster
   - C) Tema

   Respuesta Correcta: B) Clúster

10. ¿Qué tipo de archivos se almacenan típicamente en HDFS?
   - A) Archivos pequeños
   - B) Archivos grandes y fragmentados
   - C) Archivos de texto plano solamente

   Respuesta Correcta: B) Archivos grandes y fragmentados
